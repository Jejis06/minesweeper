##  Installation Steps:

1. Clone the repo &amp; open it


```bash
 git clone https://github.com/Jejis06/saper.git && cd saper
```

2. Compile the game

```bash
g++ src/main.cpp -o saper
```

3. Run it

```bash
./saper
```

<br>
